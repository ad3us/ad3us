Mastering Python With Boot.dev and Codewars

Aug 3, 2025

Python: To-go Programming Language

Recently i started to learn the Python programming language. You might say:“Ah, shit! Here we go again.”, but listen! Yes, it is annoying that everyone and everybody screams and yells that it is slow and unflexible language… but! It’s still the best language for the beginners. Some of my friends recommended me to start with C, because it’s the primary language for CS, and i tried to learn it… But, unfortunately i quit. I tried finding something easier for the beginning, and it was Python.

Coolest courses ever!

So, as a beginner, i started to find some info to learn python most effective way possible. Found some books, websites, courses, etc… but none of them was effective as the course in the Boot.dev called “Learn to code in Python”. It was very interesting journey for me and to these days, I am still using this website to learn stuff. Very interactive, fun and free! Well, not fully free. Free plan excludes interactivity, learning progress, but, still, you can see the all lessons and using you local IDE to learn some stuff.

Solve some katas :)

The second useful service is Codewars. In this website you can solve some problems by their difficulty and earn some progress. There is a scoreboard with the strongest mfs in this service and also, you can challenge them. It’s like a duel. Cool isn’t it? And it’s free!

I hope you will check them both because those services are amazing.
